# Viento de Madera

Sitio web de muebles industriales artesanales hechos a medida.

**Web en producción:** https://arbeaqua-crypto.github.io/vientodemadera/ (próximamente con dominio propio en `vientodemadera.com`)

## Estructura

```
vientodemadera-web/
├── index.html              ← Home
├── mesas-comedor.html      ← (próximamente)
├── estanterias.html        ← (próximamente)
├── mesitas.html            ← (próximamente)
├── css/
│   └── style.css
├── js/
│   └── main.js
└── images/
    ├── mesas/
    ├── estanterias/
    └── mesitas/
```

## Stack

- HTML / CSS / JS vanilla — sin frameworks
- Tipografías Google Fonts: Cormorant Garamond + Inter
- Despliegue: GitHub Pages

## Filosofía de diseño

- Sin iconos
- Paleta sobria: negro, blanco roto, acentos madera
- Fotos grandes, mucho aire, tipografía elegante
- Sin precios públicos: presupuestos personalizados por consulta
